# How To Create Popup Modal Box using HTML & CSS | Web Push Notification Design

Um projeto simples de **popup modal / notificação web** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar uma **caixa modal (popup)** e um estilo de **web push notification**, ideal para alertas e mensagens no site.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Popup modal
- Design de notificação
- Layout simples e responsivo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
